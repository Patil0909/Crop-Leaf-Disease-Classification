# Crop-Leaf-Disease-Classification
Hybrid Crop Leaf Disease Classification uses Machine Learning and Deep Learning techniques to detect plant diseases from leaf images. The system analyzes PlantVillage dataset images and predicts diseases accurately, supporting early diagnosis and improved agricultural productivity.
# Hybrid Crop Leaf Disease Classification using Machine Learning and Deep Learning

## Overview

This project focuses on automatic detection and classification of crop leaf diseases using image processing and machine learning techniques.

## Dataset

PlantVillage Dataset (Kaggle)

## Technologies Used

* Python
* OpenCV
* Scikit-learn
* NumPy
* Matplotlib
* Tkinter

## Methodology

1. Image Acquisition
2. Image Preprocessing
3. Feature Extraction
4. Random Forest Classification
5. Disease Prediction

## Results

The model classifies multiple crop diseases including Tomato, Potato, and Bell Pepper diseases.

## Future Scope

* CNN-based deep learning model
* Mobile application deployment
* Real-time disease detection
